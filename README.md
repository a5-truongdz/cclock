# cclock
A KDE clock. It uses the font used in Crystal Castles logo.

# Installation
```
$ git clone https://github.com/a5-truongdz/cclock.git
$ cd cclock
$ cp -r com.tr43212.cclock ~/.local/share/plasma/plasmoids/
```

# Configuration: Quotes
1. Edit `com.tr43212.cclock/contents/ui/main.qml`:
```
$ vi ~/.local/share/plasma/plasmoids/com.tr43212.cclock/contents/ui/main.qml
```
2. Change the `source` property to:
- `excuses`: Developer excuses
- `crystal-castles`: Crystal Castles's famous songs lyrics
